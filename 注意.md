前端开发规范与设计概要 (Frontend Specification)
1. 项目基础 (Project Basics)
技术栈：Vue 3 (Composition API) + Vite + Tailwind CSS + Axios + Chart.js

核心目标：构建 AI 智能运维平台 (AIOps) 的 Web 客户端。

设计风格：企业级、冷静、高效。

2. 视觉设计系统 (Design System)
2.1 色彩令牌 (Color Tokens)
请在 tailwind.config.js 中配置以下自定义颜色：

Brand (品牌色):

DEFAULT: #4299e1 (主按钮、激活状态)

Hover: #3182ce (悬停交互)

Light: #90cdf4 (辅助/图表虚线)

Surface (界面色):

Dark: #1a2530 (侧边栏、深色背景)

Sidebar: #2d3748 (侧边栏背景)

Card: #ffffff (内容卡片)

Background: #f7fafc (全局底色)

Status (状态色):

Success: #48bb78 (健康、通过)

Warning: #ed8936 (高负载、警告)

Error: #f56565 (故障、错误)

2.2 布局与间距
圆角 (Radius): 统一使用 rounded-lg (8px)。

间距 (Spacing): 模块间距 32px (m-8)，组件内边距 24px (p-6)。

字体: 优先使用 PingFang SC, Microsoft YaHei。

3. 页面开发规范 (Page Specs)
3.1 登录页 (Login View)
布局：左右分屏 (Split Screen)。

左侧 (45%)：深色背景 (bg-dark) + 品牌 Logo + 3个特性列表 (带绿色对勾图标)。

右侧 (55%)：白色背景 + 垂直居中表单。

交互逻辑：

输入：邮箱 (Email)、密码 (Password)。

动作：点击登录调用后端 API。

API 对接：

URL: POST /user/login (参考后端 UserController)

Payload: { "username": "...", "password": "..." }

Response: 解析 JWT 或 Session，成功跳转 /dashboard。

3.2 主界面 (Dashboard Layout)
整体结构：

Sidebar (240px, Fixed): 深色 (bg-sidebar)。包含 Logo (AIOps) 和导航菜单 (Dashboard, Monitor, Logs, Alerts, Settings)。

Header (64px, Fixed): 白色，含面包屑、全局搜索、通知铃铛、用户头像。

Content Area: 灰色底 (bg-background)，padding 32px。

核心组件：

Metric Cards (指标卡)：4列布局。展示 "系统健康度(98分)", "活跃告警(3)", "CPU负载(45%)", "预测异常(0)"。

Traffic Chart (流量图)：占据 70% 宽度。使用 Chart.js 绘制折线图，实线为实时数据 (brand.DEFAULT)，虚线为 AI 预测 (brand.light)。

Alert List (告警列表)：占据 30% 宽度。列表展示最新 Error/Warning 日志。

API 对接：

数据源 URL: POST /info/selectInfo (参考后端 InfoController)

注意：后端目前使用 POST 查询，需构造查询条件 Body。

4. 组件代码风格 (Code Style)
Vue 组件：使用 <script setup> 语法糖。

样式类：优先使用 Tailwind Utility Class，尽量不写自定义 CSS。

图标：使用 Heroicons 或 SVG 字符串。

响应式：移动端 (Mobile) 隐藏侧边栏，改为汉堡菜单；桌面端 (Desktop) 固定显示。
